#Informações sobre desafio DIO Linux Experience.

- Lembre-se de excluir diretórios, arquivos, grupos e usuários anteriores.
- Deletando usuário: 
	- userdel "usuário" obs:(usado quando usuário não tiver um diretorio).
	- userdel -r -f "usuário" obs:(usado para deletar usário por completo).

- Deletando diretórios:
	- rmdir "Dir" obs:(para diretorios vazios)
	- rmdir -p "Dir/Dir1/Dir2" obs:(para diretórios e relativos também).
	- rmdir -rf "Dir" obs:(força a remoção de subdiretórios e arquivos).

- Deletando arquivos:
	- rm "arquivo.txt" obs:(para diretorios vazios)

- Após criação do arquivo "iac.sh" lembrar de mudar a permissão "chmod 774 iac.sh)
para o scrip ser rodado com sucesso.
